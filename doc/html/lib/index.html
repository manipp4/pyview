Datacube¶

class pyview.lib.datacube.Datacube(*args, **kwargs)¶

Bases: pyview.lib.patterns.Subject, pyview.lib.patterns.Observer, pyview.lib.patterns.Reloadable

This is a hirarchical data storage class.

A datacube stores a 2-dimensional array of values. Each array len(self._table[self._index,:])) is identified by a name. In addition, you can add one or more “child datacubes” to each row of the array, thus creating a multidimensional data model.

search(**kwargs)¶

Searches for a given combination of values in the datacube. Example: datacube.search(a = 4, b = -3,c = 2,start = 0) will return the index of the first row where a == 4, b == -3, c == 2, starting at index 0. If no row matches the given criteria, search will return [].

restoreState(state)¶

Restores the state of the instrument given by “state”.

“state” can be of any type. For instruments that can store configuration data on a hard disk, “state” could be the name of the file on the disk. For other instruments, “state” could contain all relevant instrument parameters in a dictionary.
